Colour · Theme · Style reference — a warm Sunset palette across the dotfiles

🌅 Overview

This repo carries one signature look: a hand-tuned Sunset theme of warm reds, oranges and yellows over a deep ember-dark background. It is shared, hex-for-hex, across the Ghostty and Alacritty terminals and the Neovim colorscheme so the prompt, editor and shell all read as one continuous gradient. The lone exception is tmux, which runs a stock Catppuccin Mocha status line. Flip the toggle in the corner for a Sunrise light remix — the same palette re-grounded on warm paper.

💻 In Practice

How the Neovim syntax mapping (nvim/config/lua/plugins/sunset-theme.lua) paints code.

sunset.lua — nvim 
-- Sunset Theme: warm reds, oranges, yellows
local sunset = {
  bg    = "#1d1715",   -- ember dark
  accent= "#ff8c42",   -- cursor orange
}

function M.paint(colors)
  for name, hex in pairs(colors) do
    vim.api.nvim_set_hl(0, name, { fg = hex })
  end
  return true
end
