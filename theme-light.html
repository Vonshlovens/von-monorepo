Colour · Theme · Style reference — a warm Sunrise (light) take on the Sunset palette

🌄 Overview

The light companion to the Sunset reference. Same signature look — warm reds, oranges and yellows — but grounded on a warm-paper background, with the accent tones darkened just enough to stay legible against the light. The shared 16-colour palette, UI roles and per-folder breakdown are unchanged; only the surface flips. tmux still rides its stock Catppuccin Mocha status line.

💻 In Practice

How the Neovim syntax mapping (nvim/config/lua/plugins/sunset-theme.lua) paints code.

sunset.lua — nvim 
-- Sunset Theme: warm reds, oranges, yellows
local sunset = {
  bg    = "#1d1715",   -- ember dark
  accent= "#ff8c42",   -- cursor orange
}

function M.paint(colors)
  for name, hex in pairs(colors) do
    vim.api.nvim_set_hl(0, name, { fg = hex })
  end
  return true
end
